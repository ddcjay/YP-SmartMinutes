# 校史分鏡討論會議錄音(20260424)

## 摘要總覽

這是一份基於會議逐字稿整理的結構化技術會議紀錄：

### 摘要總覽
本次會議旨在討論延平中學紀念影片（校史）的分鏡規劃與敘事邏輯。核心共識為：影片將採用「剪影與光影」為視覺風格，透過「螢光/光點」象徵當時知識份子的熱血與使命感。影片敘事結構需修正時間線與史料細節，並由校方提供精確的名單、照片及文字說明，以達成敘事與史實的平衡。

### 待辦項目
| 負責人 | 事項 |
| :--- | :--- |
| **校史團隊/主任** | 提供「延平學院籌設與遷校」精確史實大紀事（包含歷年關鍵年代、變革點）。 |
| **校史團隊/主任** | 挑選影片結尾需呈現的「關鍵照片」清單，並撰寫對應的正確「圖說」。 |
| **校史團隊/主任** | 確認「延平」校名提字來源與歷史背景，並確認是否需修復（或刪除）陳澄波相關敘述。 |
| **校史團隊/主任** | 協助確認影片中出現的教師名單，評估是否全文朗讀以示尊重，或以字幕呈現。 |
| **佳穎老師** | 根據校方提供的大紀事調整分鏡時序，優化敘事邏輯（特別是復校與辦學的過渡段落）。 |
| **佳穎老師** | 調整畫面細節（如：修正國旗懸掛方式、調整顏色對比、處理照片泡泡雜訊）。 |

### 重點記錄
*   **視覺語言：** 採用黑色剪影作為角色呈現，以「天氣/光影」暗示環境壓抑（冷色調）與熱血願景（暖色調/螢光）。
*   **關鍵意象：** 
    *   以「船的剪影」象徵遣返與回台。
    *   以「油燈/螢光」象徵教育理想，從點點星光匯聚成窗戶亮光。
    *   以「稻穗/冒芽」隱喻戰後重建與後續教育工作的延續。
*   **敘事修正：** 
    *   **用語精煉：** 將「流亡四處」調整為「躲藏四處」，以符合歷史情境。
    *   **結構簡化：** 建議將「風波平息後」與「辦學奔波」的段落結合，減少影片冗餘感，優化敘事節奏。
    *   **校名起源：** 確認當時定名與提字的情境，並確認是否有真實影像支援，若無則捨棄提字畫面。
*   **內容整合：** 由於涉及大量歷史細節（補校、學院、初中等），須依據校方提供的正確年代順序進行編輯。

### 建議行動
1.  **影像史料驗證：** 針對有「泡泡」（影像瑕疵）的照片，嘗試透過修圖或 AI 技術處理，若品質不佳則需改採國圖資源或其他來源。
2.  **旁白配音策略：** 針對關鍵名言（如：非訓練什麼、我們要的是什麼樣的人），建議區分男女聲配置，增加歷史感與儀式感。
3.  **時序節奏優化：** 考慮影片總長（預估 10 分鐘），建議在不犧牲情感敘事的前提下，刪減過於細瑣的申請流程說明，保留核心精神（本職不變、初心還在）。
4.  **後續溝通：** 持續保持聯繫，待校方提供確認版史實紀要後，更新分鏡草稿進行第二次確認。
