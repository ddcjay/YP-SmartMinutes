## 4. 技術風險與緩解策略

| 風險 | 影響 | 緩解策略 |
|:---|:---|:---|
| 模型推論佔用資源 | 伺服器無 GPU 時轉錄較慢 | Faster-Whisper 已比原版快 4 倍，可選 `small` 模型 |
| 長音檔記憶體溢出 | 處理失敗 | Faster-Whisper 內建串流處理，不需一次載入全部音檔 |
| Gemini Token 長度限制 | 長逐字稿無法一次送入 | Gemini 2.0 Flash 支援 100 萬 Token，一般會議足夠 |
| 中文辨識準確度 | 專有名詞錯誤率高 | 未來加入線上校對編輯器讓使用者手動修正 |

## 5. MVP 範圍界定

### ✅ MVP 包含

- 音檔上傳（MP3/WAV/M4A/MP4）
- Faster-Whisper 語音轉文字（繁中+中英夾雜）
- AI 摘要生成（3 種預設模板：一般週會/訪談/腦力激盪）
- 摘要格式：摘要總覽 + 待辦項目 + 重點記錄 + 建議行動
- 逐字稿 / 摘要線上檢視
- 匯出下載：SRT、TXT、摘要文字、Markdown
- SSE 即時處理進度顯示

### ⏳ 未來版本

- 語者辨識 (Speaker Diarization)
- 線上校對編輯器（點擊跳轉音軌）
- PDF / Word 匯出
- 使用者帳號系統（團隊版）
- Google Drive / OneDrive 整合
- 自訂 Prompt 模板

### 啟動方式

```bash

cd backend && .\venv\Scripts\activate && uvicorn app.main:app --port 8000
```
