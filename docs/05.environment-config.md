# 環境變數與設定規範

## 1. 環境變數清單 (.env.example)

```bash
# ===== 應用設定 =====
APP_ENV=development                    # development | staging | production
APP_DEBUG=true
APP_SECRET_KEY=your-secret-key-here    # 用於 JWT 簽發，生產環境請更換

# ===== 資料庫 =====
DB_HOST=localhost
DB_PORT=5432
DB_NAME=smartminutes
DB_USER=postgres
DB_PASSWORD=your-db-password

# ===== Redis =====
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_PASSWORD=

# ===== AI 服務 =====
OPENAI_API_KEY=sk-your-openai-api-key  # Whisper API + GPT-4o 摘要
WHISPER_MODEL=medium                    # tiny|base|small|medium|large-v3
WHISPER_DEVICE=cpu                      # cpu|cuda
HUGGINGFACE_TOKEN=hf-your-token        # Pyannote 模型下載授權

# ===== 檔案儲存 =====
UPLOAD_DIR=./uploads                   # 音檔暫存路徑
MAX_FILE_SIZE_MB=500                   # 單檔最大上傳大小
FILE_RETENTION_HOURS=24                # 音檔保留時間（小時）

# ===== 前端 =====
NEXT_PUBLIC_API_URL=http://localhost:8000
NEXT_PUBLIC_WS_URL=ws://localhost:8000
```

## 2. Docker Compose 配置概要

```yaml
# docker-compose.yml (開發環境)
services:
  db:
    image: postgres:16-alpine
    ports: ["5432:5432"]
    environment:
      POSTGRES_DB: smartminutes
      POSTGRES_USER: postgres
      POSTGRES_PASSWORD: ${DB_PASSWORD}
    volumes:
      - pgdata:/var/lib/postgresql/data

  redis:
    image: redis:7-alpine
    ports: ["6379:6379"]

  backend:
    build: ./backend
    ports: ["8000:8000"]
    env_file: .env
    depends_on: [db, redis]
    volumes:
      - ./backend:/app
      - ./uploads:/app/uploads

  celery-worker:
    build: ./backend
    command: celery -A app.tasks worker -l info
    env_file: .env
    depends_on: [db, redis]

  frontend:
    build: ./frontend
    ports: ["3000:3000"]
    environment:
      NEXT_PUBLIC_API_URL: http://localhost:8000
    depends_on: [backend]

volumes:
  pgdata:
```

## 3. 安全注意事項

> [!CAUTION]
> - `.env` 檔案 **必須** 加入 `.gitignore`，禁止提交至版本控制
> - 生產環境的 `APP_SECRET_KEY` 必須使用隨機生成的強密碼
> - `OPENAI_API_KEY` 與 `HUGGINGFACE_TOKEN` 為敏感憑證，禁止外洩
